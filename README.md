🛒 Online Grocery Application

A full-stack web application for managing and purchasing groceries online.
This project includes a user frontend, admin panel, and a Spring Boot backend API.

🚀 Features
👤 User Side
Browse grocery items
View product details
Add to cart
Place orders
User authentication (Login/Register)
🛠️ Admin Panel
Add new grocery items
View product list
Manage orders
⚙️ Backend (Spring Boot)
REST APIs for all operations
JWT-based authentication
Cart & Order management
Database integration
